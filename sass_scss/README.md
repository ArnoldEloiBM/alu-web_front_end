# Sass & Scss

This directory contains Sass/SCSS files demonstrating various features and concepts.

## Files

- `0-debug_log.scss` - Demonstrates @debug directive to print "Hello world" in debug output
- `1-color_variable.scss` - Color variable example
- `2-color_variables.scss` - Multiple color variables
- `3-nested_tag.scss` - Nested tag selectors
- `4-nested_class.scss` - Nested class selectors
- `5-nested_child.scss` - Nested child selectors
- `6-nested_hover.scss` - Nested hover states
- `7-nested_deeper.scss` - Deeper nesting examples
- `8-mixin_margins.scss` - Mixin examples for margins
- `9-extend_list.scss` - @extend directive examples
- `10-colors.scss` - Color definitions
- `10-import_colors.scss` - Importing color files
- `11-photos.scss` - Photo styling
- `11-loop_photos.scss` - Loop examples for photos
- `12-loop_header.scss` - Loop examples for headers
